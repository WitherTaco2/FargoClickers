Fargo`s Mod Clickers is addon for Fargo's mod that adds new boss clickers, accessories and compatibilities

Features:
1. Clickers that drops by mod's bosses, like Lifelight or Cursed Coffin
2. Enchantments, Force from clicker armors has unique effect (or upgraded version of armor set bonus) than dumb copy armor set bonus
3. Desktop Computer how has neaby effect like Ammo Box
4. Has compatibility with Calamity Clickers while installed Calamity - Fargo's Souls DLC
